# Collect Title

<p align="center">
  <img src="/src/main/resources/static/assets/logo-collect-title-dark.svg">
</p>

O objetivo do projeto **Collect Title** é facilitar o gerenciamento/controle de títulos.

## :wrench::hammer: Tecnologias utilizadas 

- **Java**
- **Spring Boot**
- **Spring Data JPA**
- **Thymeleaf**
- **MySQL**
- **Apache Tomcat**
- **Maven**
